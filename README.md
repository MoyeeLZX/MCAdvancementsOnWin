# 介绍
## 适用场景
+ 一个正常运转的**Windows10及更新的系统**。~（Windows7没试过）~
## 功能
当程序在运行的时候，会检查Windows上当前用户正在运行的进程以及窗口标题。若达成某个条件则触发像[Minecraft Java Editon](https://zh.minecraft.wiki/w/Java%E7%89%88)中的[进度](https://zh.minecraft.wiki/w/%E8%BF%9B%E5%BA%A6)一样，同时播放[音效](https://zh.minecraft.wiki/images/transcoded/Challenge_complete.ogg/Challenge_complete.ogg.mp3)。
